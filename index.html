En mi corto recorrido dentro de la programación web he aprendido términos básicos, que es un full stack, los elementos de una página, código y estructura de HTML y CSS, atajos de VSC, plataformas, programas para el diseño, la programación, la modelación, la maquetación, el almacenamiento de mis proyectos y un sin fin de elementos y material multimedia que me permitirán subir mi nivel en la programacion web pese a que aun quede un gran camino por recorrer.

En esta asignatura he aprendido a utilizar algunas herramientas que me permiten y permitirán tener una gran facilidad, comodidad, velocidad y almacenamiento al realizar páginas, algunas de estas herramientas son:

- El editor de código fuente Visual Studio Code desarrollado por Microsoft para Windows, Linux y macOS. Incluye soporte para la depuración, control integrado de Git, resaltado de sintaxis, finalización inteligente de código, fragmentos y refactorización de código. También es personalizable, por lo que los usuarios pueden cambiar el tema del editor, los atajos de teclado y las preferencias. Es gratuito, de código abierto. y ofrece extensiones con distinto atributos que le permiten al usuario ser más rápido, creativo, entre otras cosas, al crear páginas web.

- GitHub es un sistema de gestión de proyectos y control de versiones de código, así como una plataforma de red social diseñada para desarrolladores. Permite trabajar en colaboración con otras personas de todo el mundo, planificar proyectos y realizar un seguimiento del trabajo. Es también uno de los repositorios online más grandes de trabajo colaborativo en todo el mundo.
